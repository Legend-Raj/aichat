## Running on Localhost

- Clone this repository

```bash
git clone https://github.com/Legend-Raj/aichat/
```

- Install dependency library

```bash
npm i
```

- Copy `.env.example` to `.env` and fill it with your API

```bash
cp .env.example .env
```

- Run Application

```bash
npm i
npm start
```
